# Flow Log

Small private-feeling water, pee, and #2 tracker built as a static web app.

## Privacy model

- The app code can be hosted publicly.
- Your logs are stored in your browser on your device.
- Export backups are optional and stay wherever you save them.
- Hosting providers can still see ordinary site traffic metadata.

## Features

- Quick log buttons for water, pee, and #2
- Today dashboard
- Seven-day histogram
- Browse all recorded days
- Export and import JSON backups
- Offline support after first load

## Local development

Run a static server from this folder, for example:

```bash
python3 -m http.server 8000
```
